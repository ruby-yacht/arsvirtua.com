How To Connect

you can connect in many ways, the essential details are listed at the end of this page:

connect via mud client

connect via telnet client (on computer or phone)

connect via http bridge which seems to only work properly with firefox

you can connect by downloading a telnet client, telnet is an ancient network client that can talk to servers in plain text over a network connection. You can find telnet clients and mud clients. I personally use Atlantis.

if you want an account you can go to http://bit.ly/look-art-login and request a login

If you are unable to connect via telnet there is an experimental web to telnet site hosted by Mosha. The serveris supported by donations and may occasionally go down. Instructions are available to help you log in
